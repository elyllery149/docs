# You Don't Know JS Yet (book series) - 2nd Edition

This is a series of books diving deep into the core mechanisms of the JavaScript language. This is the **second edition** of the widely acclaimed book series.

<Note>
**To read more about the motivations and perspective behind this book series, check out the [Preface](/preface).**
</Note>

If you're looking for the previous **first edition** books, they can be found in the [original repository](https://github.com/getify/You-Dont-Know-JS/blob/1st-ed/README.md).

## Book Titles

I recommend reading the **second edition** books in this order:

<CardGroup cols={2}>
  <Card title="Get Started" icon="rocket" iconType="duotone" color="#fc9304" href="https://leanpub.com/ydkjsy-get-started">
    The foundational book that introduces you to JavaScript's core concepts and gets you started on your journey.
  </Card>
  
  <Card title="Scope & Closures" icon="brackets-curly" iconType="duotone" color="#f46404" href="https://leanpub.com/ydkjsy-scope-closures">
    Deep dive into lexical scope, function vs. block scope, hoisting, and closure patterns.
  </Card>
  
  <Card title="Objects & Classes" icon="cube" iconType="duotone" color="#fcb47c" href="https://leanpub.com/ydkjsy-unbooks">
    Explore prototypes, object patterns, and the class system (draft stable).
  </Card>
  
  <Card title="Types & Grammar" icon="brackets-square" iconType="duotone" color="#fc9304" href="https://leanpub.com/ydkjsy-unbooks">
    Understanding JavaScript's type system and language grammar (rough draft).
  </Card>
</CardGroup>
